# SuperGopher - Game

## About the Game

SuperGopher is a fun 2D platformer game where you take control of a character and navigate through various levels filled with challenges, enemies, coins, and power-ups. Collect coins, avoid dangerous obstacles, and reach the treasure chest to complete the game!

## How to Run the Game

1. **Download and Extract**: Clone or download this repository to your computer
2. **Navigate to the project folder**: `cd SuperGopher`
3. **Open the game**: Open `MainMenu.html` in your web browser
4. **Start playing**: Click on "Play" to begin your adventure

**Requirements**: A modern web browser (Chrome, Firefox, Safari, or Edge)

## Game Overview

<img src="src/img/gamemain.png" alt="SuperGopher Gameplay" width="400">

In SuperGopher, you embark on an exciting platformer adventure where:

- Navigate through challenging levels filled with platforms and obstacles
- Collect coins to earn points and progress through the game
- Avoid dangerous enemies and environmental hazards
- Use power-ups strategically to help you on your journey
- Reach the treasure chest to complete the game with two possible endings

## How to Play

### Character Movement
- **A** - Walk left
- **D** - Walk right
- **SPACE** - Jump
- **SHIFT** (Hold) - Run (doubles your speed and jump force)

### Game Enemies and Objects
- **Clouds with Faces**: Be careful! Touching one will cost you a life. You have 3 lives per game.
- **Coins**: Collect them for 10 points each
- **Lucky Blocks**: Touch them to gain 50 points. You can collect from the same block up to 3 times
- **Treasure Chest**: Reach it with more than half the coins collected to gain 100 points and complete the game (2 possible endings)
- **Power-ups**: Use various power-ups to help you complete your objective
